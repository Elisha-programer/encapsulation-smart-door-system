# Encapsulation Smart Door System

## Overview
This project demonstrates the concept of encapsulation in Python by developing a Smart Door Access System for UMaT administrative staff.

## Features
- Public attribute: `s_name`
- Private attribute: `__access_code`
- Uses the `@property` decorator to safely access the private access code.
- Validates the access code before updating it.
- Displays staff information.

## Concepts Used
- Encapsulation
- Private Attributes
- Properties (`@property`)
- Property Setter
- Input Validation
